# Registro de Testes de Usabilidade

Após realizar os testes de usabilidade, obtém-se um relatório a partir das análises realizadas. O Registro de Testes de Usabilidade é um relatório que contém as evidências dos testes e relatos dos usuários participantes, baseado no Plano de Testes de Usabilidade desenvolvido para os casos de uso desta etapa.

O teste foi realizado com 14 pessoa, as quais cumpriram as seguintes tarefas:

- Cadastro de Usuário;
- Cadastro de Instituição;
- Login;
- Agendamento de Atendimento Psicológico;
- Reserva de Alojamento;
- Navegação.

## Dados coletados

Após a coleta das informações concluiu-se:

### Qualidade do site
Por meio das perguntas discursivas, principalmente, observa-se que apesar das melhorias que podem ser feitas, o site é bem feito e de fácil entendimento.

### Conclusão de tarefas
- 100 % dos participantes concluíram as tarefas propostas.

### Sucesso na realização das tarefas
- 100 % dos participantes sentiram-se satisfeitos ao realizarem as tarefas propostas.

### Facilidade na realização de tarefas
- 42,86 % dos participantes classificaram a realização de tarefas como "Fácil" e 57,14 % classificaram como "Muito Fácil".
